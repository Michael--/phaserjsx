# Changesets

This directory contains changeset files that describe changes to packages in this monorepo.

## How to use

1. Make your changes
2. Run `pnpm changeset` to create a new changeset
3. Describe your changes following the prompts
4. Commit the changeset file along with your changes
5. On merge to main, the release workflow will handle versioning and publishing

See https://github.com/changesets/changesets for more information.
